Git基本操作

发表于 2017-01-15 | 分类于 版本控制

Git 是目前最流行的版本管理系统

Windows系统安装和使用

安装后会出现三个工具-Bash,-CMD,-GUI，只用-Bash就行

在你想作为本地仓库的文件夹上（或进入文件夹空白处）鼠标右键，选择Git Bash here

git clone

​ <本地路径>省略即-Bash当前路径
