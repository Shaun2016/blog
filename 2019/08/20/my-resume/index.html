脑与认知计算实验室 量表测评系统 （ 2019年1月 ~ 2019年6月 ）

我在此项目中负责在线问卷的制定与展示，使用 matlab 导出的 jar 包对问卷答案做测评；由于问卷格式不固定，无法用关系型数据库保存问卷，我使用类似编写 Markdown 文档的形式，制定了一些语法规则，在后端将录入的字符串解析成问卷对象，再将对象的 json 数据保存在 mysql 中；对问卷答案测评的过程比较耗时，我通过创建线程的方式去执行这个过程，用线程池管理创建的线程，避免主线程的阻塞；测评结果写入数据库中，通过前端刷新页面查看测评结果；

主要使用 Springboot，Jpa，Thymeleaf 开发

加入隐藏关系的属性社区搜索算法的研究 （2018年9月 ~ 2019年1月）

普通的社区搜索算法是在原图中通过 k-core 或 k-truss 结构寻找评分最高的社区，考虑到图信息不完整性，可能存在有联系的结点之间没有边，或者在未来很可能建立边的情况；我通过神经网络模型进行网络表示学习，使用表示向量做链接预测从而找到隐藏的边，加入隐藏关系后使得搜索的社区结果有更高的查全率和F值

动力学院英文主页 （2017年11月 ~ 2017年12月）

使用 WebPlus 平台和 Bootstrap 框架为动力学院制作的响应式的英文主页，WebPlus 是一个拖拽组件式的 CMS 平台，很多组件代码是拖拽后系统生成的，使得页面效果与预期不符；我通过对生成的 css 重写，js 修改 DOM 结构，达到预期的效果: http://pnec.hrbeu.edu.cn/en3/

基因行业团体标准网 （2017年9月 ~ 2017年12月）

独自开发包括 CMS 系统和项目申报系统的网站；制作 WordPress 模板，使用 WordPress 搭建网站的 CMS 部分；使用 laravel 框架开发项目申报系统，实现了立项，审批，结题，审核一系列流程，包括邮箱登陆注册，文档上传下载，组委员审批项目，人员管理等功能，最后部署 nginx 服务器上
