自己实现一个进度条

发表于 2020-03-21 | 分类于 python

在控制台打印进度，统计进度，已有时间，剩余时间，适应性还不是很好；可以自定义进度条的长度和进度的总数，两者是独立的；用户可以直接传入想显示的进度，也可以将当前进度加上一个数值；

存在问题：

  • 在 mac 系统中\r不会讲光标退回行首，无法覆盖之前的打印信息，而是会换行
  • 无法自义定是否显示进度条的名称，Cost，Left功能，以及修改这些文字
