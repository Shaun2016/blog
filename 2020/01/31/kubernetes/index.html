K8s的安装

环境

Master : 192.168.137.100

Slave1 : 192.168.137.101

Slave2 : 192.168.137.102

下面的操作为仅在Master中的操作，其余两台从节点也要执行相应的操作

  1. 设置主机名和时区

    timedatectl set-timezone Asia/Shanghai

    hostnamectl set-hostname master

  2. 配置主机名对应的IP

    vi /etc/hosts

  3. 关闭防火墙

安装kubeadmin

  1. XFTP上传kubernetes离线安装文件

  2. 在每个节点上安装Docker

  3. docker info | grep cgroup查看Cgroup Driver:cgroupfs

  4. tar -zxvf kube114-rpm.tar.gz

  5. cd kube114-rpm

  6. yum localinstall -y *.rpm

  7. 关闭交换区swapoff -a，vi /etc/fstap注释带有swap一行

  8. 配置网桥

  1. 指令随记

cat >file表示重定向输出到file中，用户输入内容后按ctrl+c结束编写内容并将内容输出到file中，注：会覆盖file中原始的内容；

cat >> file是在file 中追加内容

上述的两个指令无法在脚本执行中使用，因为会阻塞等待用户输入，需要使用下面的指令：

cat <<EOF > file创建文件并写入内容，输入完成后以EOF结束，其中EOF只是一个字符串标记，可以随意指定，只是默认的写法是EOF；在脚本中想向文件中写入内容时使用，cat <<EOF >> file表示追加内容

echo 12345 > 1.txt

cat < 1.txt > 2.txt

在上面的两条指令中，>表示输出流（>覆盖>>追加），<表示输入流，如cat < 1.txt > 2.txt表示将1.txt的内容做为输入，输出到2.txt中
